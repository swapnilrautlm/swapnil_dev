Daily rhythm

Summer weekday — English leads now

  • English (~2.5–3.5h): grammar/expression block first (Khan U10/U12 + Wren + Bank), then a reading block (Khan + Bank, Hard). This is where the points are.
  • Math (~1–1.5h, maintenance): one short hard set on the rotating domain (mostly Data & Geometry), Desmos + precision; full timed module only before tests.
  • ~4–5h most days. If fading, drop the math set, not the English blocks.

Weekends & school weeks

  • Weekends: light (2h max) — error-log review, vocabulary, or rest. Practice tests take the Saturday morning.
  • From Sep 2 (school 7–3): ~1h+1h in the evening, biased to English — mostly review and short timed sets.
  • Fri Sep 11: rest and logistics before exam day. The exact per-day skills live in the day-by-day calendar.

The three resources

Learn → drill → reinforce. Each day pairs a Khan Academy lesson (to learn the skill) with a Question Bank set (to drill it), and — on grammar days — a Wren & Martin chapter (to reinforce it). The exact per-day assignments live in the day-by-day calendar; this is how to use each tool.

Khan Academy — learn

Math (v2-sat-math) is tiered Foundations → Medium → Advanced per domain — climb all three for Advanced Math, skip Foundations for Algebra. Reading & Writing (sat-reading-and-writing) is by domain and skill. Watch every video, follow along, and do the practice to "level up."

Question Bank — drill

  1. Open Create a Question Set. Assessment = SAT; Section = Math or Reading and Writing.
  2. Tick the Domain for the day, then Search.
  3. Add Filters → Skill and Difficulty (bars: 1 = easy, 2 = medium, 3 = hard).
  4. Select ~10–15 questions, Export a PDF with answers & explanations.
  5. Mark every miss and log it to the Google Sheet to re-attempt later. 1,756 Math and 1,688 R&W official questions — no repeats needed.

Wren & Martin — reinforce grammar

Powers Standard English Conventions and Expression of Ideas: punctuation & clauses for Boundaries (Bk2 Ch26; Bk1 Ch3; Bk2 Ch1–4), agreement / pronouns / tenses / modifiers for Form & Sense (Bk2 Ch17–21; Bk1 Ch24–26), conjunctions & synthesis for Transitions and Rhetorical Synthesis (Bk1 Ch40–41; Bk2 Ch12–14, 22). Its clause-analysis also helps parse dense reading passages.

Master Desmos — until it's reflex

The built-in Desmos calculator is available in both math modules — there is no no-calculator section. On hard or messy questions, graphing is faster and safer than algebra. Practice on the exact calculator inside Bluebook, not just desmos.com, so the moves are muscle memory by exam day.

GoalWhat to do in DesmosGreat for
Solve an equationType y = f(x); click where it crosses the x-axisroots, zeros, "solutions", x-intercepts
Solve two equations / a systemType both on separate lines; click each intersection point for its coordinateslinear or nonlinear systems
Count solutionsGraph both; count the intersection points"how many solutions does the system have?"
Vertex / max / minGraph the parabola; click the turning pointvertex form, maximum/minimum, axis of symmetry
InterceptsClick where the curve meets each axisy-intercept (x=0), x-intercepts
Evaluate a functionDefine f(x)=…, then type f(3) on a new line"what is f(3)?", checking values
Add a sliderType an equation with a letter, e.g. y = ax² + 3; tap "add slider" and drag"for what value of k…", matching a condition visually
Line / curve of best fitEnter data as a table (x₁, y₁), then type y₁ ~ m x₁ + b using the tilde ~. Exponential: y₁ ~ a b^x₁two-variable data, "which equation models the data"
StatisticsPut data in a list, then mean(L), median(L), stdev(L), total(L)one-variable data: center & spread
InequalitiesType y < mx + b; Desmos shades the region. Graph several for a systemlinear inequalities, feasible region
CirclesType (x−h)² + (y−k)² = r²center, radius, intersections
Tables & restrictionsAdd a table for x-values; add {a < x < b} to bound a graphplotting points, piecewise contexts

Habits that make it reflex

  • Snap decision each question: messy numbers, multi-step, "how many solutions", or best-fit → graph it. One-step → do it by hand (graphing can be slower for the trivial ones).
  • Type cleanly: watch parentheses, and use the arrow keys to step out of a fraction or exponent before typing the next part.
  • One equation per line — each Enter starts a new expression.
  • Lost the intersection? Scroll/pinch to zoom out, or set the window with the wrench icon.
  • Drill it: in each phase, spend one block re-solving a question set entirely in Desmos until the moves are automatic.
